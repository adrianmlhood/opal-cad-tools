Opal CAD Tools

AutoCAD design tools for the Opal team

Version 1.0.0

This adds the O-Suite and LayerKit commands to AutoCAD. Once installed, the tools load by themselves every time you open AutoCAD. Installing takes about a minute and does not need administrator rights.

Install in two steps

  1. 1
    Double-click Install Opal CAD Tools.bat
    A small window opens, copies the tools, and closes. That is the whole install.
  2. 2
    Restart AutoCAD
    When it finishes loading, the command line shows: Opal CAD Tools loaded.

Using the tools

  1. O
    Type O for the toolbox
    A window of labeled buttons, grouped Draw, Layers, and Setup. Click one to run it.
  2. ?
    Type OHELP for the full list
    Every command with a one-line description, printed to the command line.

Update or remove

Update: run Install Opal CAD Tools.bat from a newer copy.
Remove: double-click Uninstall Opal CAD Tools.bat.

Good to know

Installs only for your Windows account.
Nothing is placed outside AutoCAD's add-in folder.
No administrator rights required.
The tools are plain AutoCAD scripts in a trusted folder.
